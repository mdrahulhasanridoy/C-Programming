### <mark>1.1 Environment Setup</mark>

* 🪟 Setup
    
    * Download & Install [VS Code](https://code.visualstudio.com/)
      
      ![image](https://github.com/mdrahulhasanridoy/C-Programming/assets/99040899/24f36f03-293a-4ad7-a5a5-4e8a34c6577e)

    * Download & Install All Basic Packages Of [MinGW](https://sourceforge.net/projects/mingw/)

      ![image](https://github.com/mdrahulhasanridoy/C-Programming/assets/99040899/6a40b3da-0224-4980-bb81-d11e56e9feee)

    * Copy MinGW bin path ➡️ Go to system environment variable ➡️System Variable➡️ Add MinGW bin path.

      ![image](https://github.com/mdrahulhasanridoy/C-Programming/assets/99040899/d4258109-9423-4476-b429-5b533409c201)
        
    * Open Vs Code & Install Extensions
            
            1. C/C++
                
            2. C/C++ Extension Pack
                
            3. Code Runner
                

Now your VS Code Setup is ready. You can start coding in VS Code for C programming. If you face any problem, try to solve it by yourself. It will help you to learn more.
